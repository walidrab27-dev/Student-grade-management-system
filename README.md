# 🎓 Student Grade Management System

A simple **C# Console Application** built to practice **Object-Oriented Programming (OOP)** fundamentals and working with collections in .NET.

This project allows users to manage students and their grades through a simple console menu while focusing on clean code and core C# concepts.

---

## 📌 Features

* ➕ Add new students
* 📝 Assign grades for different subjects
* 📊 Calculate student average grade
* 👨‍🎓 Display student information
* 🔍 Search students by ID
* 📚 Store grades using `Dictionary<string, double>`
* 🗂️ Manage students using `List<Student>`
